# GXTRO-Forge

GXTRO-Forge 是一款輕量級影音剪輯工具，整合 VLC 預覽與 FFmpeg 編碼技術，旨在提供流暢、精準的剪輯體驗。

## 核心功能

- **高效預覽**：基於 VLC 核心，支援 4K 即時流暢預覽。
- **精準編輯**：雙軌時間軸管理，支援毫秒級修剪與一鍵分割。
- **多工導出**：支援 MP4, MKV, AVI, GIF 等主流格式輸出。
- **專屬標識**：內建自定義浮水印系統與品牌定位。
- **智慧安裝**：提供專用的網頁安裝器與版本自動校驗。

## 快速開始

1. 從 [GitHub Releases](https://github.com/appy002255/Gxrto-Forge/releases) 下載 `GXTRO_Forge_Installer.exe`。
2. 執行安裝程式並依照引導完成佈署。
