# GXTRO-Forge

GXTRO-Forge 是一款輕量級影音剪輯工具，整合 VLC 預覽與 FFmpeg 編碼技術，旨在提供流暢、精準的剪輯體驗。

**目前版本：`v0.2.00 beta`**

## 核心功能

- **即時預覽**：基於 VLC，支援順暢的畫布預覽。
- **無限多軌**：支援影片、音軌、圖片、疊加 (PiP) 與字幕的無限軌道拖拉編輯。
- **AI 字幕**：內建 Faster-Whisper，一鍵全自動語音轉字幕 。
- **進階畫布**：支援畫布拖拉縮放、旋轉及字幕樣式微調。
- **多格式匯出**： FFmpeg 編碼核心，支援圖層混合，匯出 MP4, MKV, GIF 等。
